## рҹҡҖ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/website-GTA-VI.git
   cd website-GTA-VI
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**

   Navigate to `http://localhost:5173` to view the website.
